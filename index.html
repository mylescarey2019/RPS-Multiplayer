Welcome to Gif-Break

To find Gifs use the search control to enter a term, such as 'cat', then click 'Search'

Ten Gifs will be retrieved. Use the left/right arrows on the Gif to move to the previous/next Gif.

Search terms are saved and can be re-selected using the drop-down list

Click clear button to remove all saved searches.

For small devices the control can be shown/hidden using the expand button
